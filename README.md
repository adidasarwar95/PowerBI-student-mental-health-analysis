# Power BI Project – Student Social Media & Mental Health Analysis

**Course:** IBM Data Analyst Program – Career247 (Batch 13)
**Tool Used:** Microsoft Power BI Desktop

---

## Project Overview
Built a 6-page interactive Power BI report analyzing the impact of
social media usage on students' mental health, academic performance,
lifestyle, and relationships. Dataset contains 705 students across
multiple countries and academic levels.

---

## Report Pages

| Page | Focus |
|------|-------|
| Executive Overview | KPI cards, gender split, addiction score by academic level |
| Mental Health & Lifestyle | Health bands by country, addiction vs mental health scatter, sleep by age |
| Academic Impact | Academic impact by platform, age slicer, usage by academic level |
| Relationships & Conflicts | Relationship status distribution, conflict level analysis |
| Interactive Story View | Bookmarked views — Gender View & Academic Level View |
| Student Profile | Full student data table with all attributes |

---

## Key Metrics
- Total Students Analyzed: **705**
- % Academically Affected: **64%**
- Average Sleep Hours Per Night: **6.87**
- Average Daily Usage Hours: **4.92**
- High School students show the highest average addiction score

---

## Visuals Used
- KPI Cards

- Bar Charts (horizontal & vertical)
